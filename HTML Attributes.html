1. id: A unique identifier for the element.
2. class: Specifies one or more class names for the element, used for styling.
3. src: Specifies the URL of an image, script, or other resources.
4. href: Specifies the URL of a link.
5. alt: Provides alternative text for an image if it cannot be displayed.
6. title: Specifies extra information about an element, displayed as a tooltip.
7. style: Used to apply CSS styles directly to an element.
8. type: Specifies the type of input element or the type of button, script, etc.
9. value: Specifies the value of an input element.
10. name: Specifies the name of an input element, used in form submissions.
11. placeholder: Provides a short hint that describes the expected value of an input field.
12. checked: Indicates whether an input element (checkbox or radio button) is selected.
13. disabled: Disables an input element, making it uneditable.
14. readonly: Specifies that an input field is read-only.
15. maxlength: Limits the number of characters allowed in an input field.
16. autocomplete: Specifies whether an input field should have autocomplete enabled.
17. action: Specifies the URL to send form data to when a form is submitted.
18. method: Specifies the HTTP method (GET or POST) to use when sending form data.
19. target: Specifies where to open the linked document (e.g., `_blank` for a new tab).
20. data- (custom data attributes): Allows you to store extra information on an element (e.g., 'data-id', 'data-role')
