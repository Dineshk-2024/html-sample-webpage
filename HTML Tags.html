1. html - Defines the root of an HTML document
2. head - Contains meta-information about the document
3. title - Defines the title of the document
4. body - Defines the document's body
5. h1 to h6 - Defines HTML headings
6. p - Defines a paragraph
7. a - Defines a hyperlink
8. img - Defines an image
9. ul - Defines an unordered list
10. ol - Defines an ordered list
11. li - Defines a list item
12. div - Defines a division or section
13. span - Defines a section in a document
14. table - Defines a table
15. tr - Defines a row in a table
16. td - Defines a cell in a table
17. th - Defines a header cell in a table
18. form - Defines an HTML form for user input
19. input - Defines an input control
20. button - Defines a clickable button
